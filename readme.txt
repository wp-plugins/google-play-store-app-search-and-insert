== Description ==

This plugin help you search and insert android app info from Google Play Store to content very quickly.

Feature:

- Search android app info from Google Play Store and insert to content
- Change title and size before insert

== Installation ==

1. Unzip the google-play-store-app-search-and-insert.zip
2. Copy google-play-store-app-search-and-insert folder to wp-content/plugins
3. Go to Plugins/Installed Plugins, find Google Play Store App Search And Insert and click Active
4. Enjoy!
